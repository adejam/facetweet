### Development (Running locally)

- Clone the project

```bash
git clone https://github.com/user/html-css-template.git

```

- Install Dependencies

```bash
npm install
```

- Compiles and hot-reloads for development

```
npm run serve
```

- Compiles and minifies for production

```
npm run build
```

- Lints and fixes files

```
npm run lint
```
